Memori

Memori adalah perangkat atau sistem yang digunakan untuk menyimpan data dan informasi sehingga dapat digunakan secara langsung oleh komputer atau perangkat elektronik digital lainnya. Memori berfungsi sebagai tempat dimana program, instruksi, dan data yang sedang diproses disimpan sementara agar dapat diakses dengan cepat oleh prosesor. Tanpa memori, komputer tidak akan mampu menjalankan aplikasi maupun melakukan perhitungan karena prosesor membutuhkan ruang kerja yang cepat dan efisien.

Istilah memori sering kali digunakan sebagai sinonim dari penyimpanan utama atau memori utama (main memory), yaitu komponen penyimpanan yang bersifat volatil dan bekerja secara langsung dengan CPU, seperti RAM (Random Access Memory). Berbeda dengan penyimpanan sekunder seperti hard disk atau SSD, memori utama memiliki kecepatan yang jauh lebih tinggi namun biasanya kapasitasnya lebih kecil.

Dalam bahasa Inggris, terdapat istilah lama atau kuno yang juga digunakan sebagai padanan kata memori, yaitu “store”, yang merujuk pada tempat menyimpan data. Walaupun istilah ini jarang digunakan dalam konteks modern, maknanya tetap sama yaitu sebagai media penyimpanan data untuk keperluan pemrosesan.

Secara keseluruhan, memori adalah elemen penting dalam sistem komputer yang memastikan proses berjalan cepat, efisien, dan terkoordinasi, karena menjadi penghubung utama antara CPU dan data yang sedang diproses.

Jenis-jenis Memori

Memori primer (utama)

Memori primer adalah memori yang berhubungan langsung dengan CPU, memori sekunder, dan memori cache. Memori ini digunakan untuk menyimpan program dan data yang sedang aktif diproses oleh CPU. Ketika komputer menjalankan aplikasi, data akan dimuat ke memori primer agar dapat diakses dengan cepat oleh prosesor.

Memori primer memiliki kecepatan akses yang sangat tinggi, tetapi di sisi lain harganya relatif mahal dan kapasitasnya lebih kecil dibandingkan memori sekunder. Karena itu, memori primer dioptimalkan untuk kecepatan, bukan kapasitas.

Fungsi utama memori primer

  1. Menyimpan data sementara selama program berjalan
  2. Menyediakan ruang kerja untuk CPU
  3. Menghubungkan CPU dengan penyimpanan lain agar proses lebih cepat

Contoh memori primer

  1. MRAM (Random Access Memory)
  2. ROM (Read Only Memory)
Memori Sekunder (Secondary Storage)

Memori sekunder adalah memori yang digunakan sebagai penyimpanan jangka panjang dengan kapasitas besar. Berbeda dengan memori primer, memori sekunder tidak dapat diakses langsung oleh CPU. Jika CPU membutuhkan data dari memori sekunder, data tersebut harus dipindahkan terlebih dahulu ke RAM, baru kemudian diproses. Memori sekunder umumnya memiliki kapasitas besar dan harga lebih murah dibandingkan memori primer, tetapi kecepatan aksesnya lebih lambat.

Fungsi memori sekunder

  1. Menyimpan data secara permanen
  2. Menyimpan file berukuran besar seperti video, aplikasi, backup, dan dokumen
  3. Menjadi tempat penyimpanan sistem operasi dan program

Contoh memori sekunder

  1. Hard Disk Drive (HDD)
  2. Solid State Drive (SSD)
  3. Flash Drive / USB
  4. CD/DVD
  5. Memory Card (SD Card, MicroSD)
Memori Cache

Memori cache adalah memori kecil berkecepatan sangat tinggi yang biasanya berada di antara CPU dan RAM atau bahkan menyatu langsung dengan prosesor. Tujuan utama cache adalah untuk meningkatkan kinerja CPU dengan menyimpan data dan instruksi yang sering digunakan sehingga CPU tidak perlu mengambil data dari RAM yang lebih lambat. Semakin tinggi level cache, biasanya ukurannya lebih besar tetapi kecepatannya sedikit lebih rendah.

Level memori cache

  1. L1 Cache – paling kecil dan tercepat, berada langsung dalam inti CPU
  2. L2 Cache – lebih besar, sedikit lebih lambat
  3. L3 Cache – paling besar, mendukung beberapa inti CPU

Fungsi memori cache

  1. Mengurangi waktu akses CPU
  2. Menjembatani perbedaan kecepatan antara CPU dan RAM
  3. Mempercepat eksekusi instruksi berulang
Memori Register

Memori register adalah memori paling kecil sekaligus paling cepat yang dimiliki komputer. Register terdapat langsung di dalam CPU dan digunakan untuk menyimpan data yang sedang diproses dalam waktu sangat singkat. Karena ukurannya kecil, register hanya menyimpan data dalam bentuk bit tertentu, misalnya 16 bit, 32 bit, hingga 64 bit tergantung arsitektur CPU.

Fungsi memori register

  1. Menyimpan instruksi yang sedang diproses CPU
  2. Menyimpan alamat memori atau hasil perhitungan sementara
  3. Mengirim dan menerima data dengan kecepatan sangat tinggi

Contoh register

  1. Program Counter
  2. Instruction Register
  3. Accumulator
  4. Stack Pointer

Sejarah Memori

Perkembangan memori komputer berlangsung melalui berbagai tahapan panjang sejak awal kemunculan komputer modern. Setiap generasi memori selalu didorong oleh kebutuhan untuk menyediakan kapasitas yang lebih besar, kecepatan lebih tinggi, ukuran lebih kecil, dan efisiensi energi yang lebih baik. Berikut penjelasan evolusi memori dari generasi awal hingga teknologi memori modern.

Generasi Awal: Magnetic Core Memory (1950–1970-an)

Pada masa ini, komputer menggunakan magnetic core memory, yaitu memori yang terbuat dari cincin ferit kecil yang dapat menyimpan bit dengan mengubah arah magnetnya. Meskipun begitu, teknologi ini menjadi landasan komputer era awal, seperti UNIVAC dan IBM System/360.

Generasi Transisi: Magnetic Drum & Delay Line Memory

Sebelum RAM modern lahir, komputer menggunakan memori berbasis mekanik dan gelombang. Magnetic drum memory: semacam drum berputar dengan lapisan magnetik. Delay line memory: menggunakan gelombang suara dalam medium (misalnya raksa). Teknologi ini akhirnya ditinggalkan ketika RAM elektronik ditemukan.

Generasi RAM Awal: SRAM & DRAM (1970–1980-an)

Perkembangan besar terjadi saat semikonduktor mulai digunakan dalam memori komputer.

SRAM (Static RAM)

  1. Kecepatan tinggi.
  2. Tidak perlu direfresh.
  3. Digunakan sebagai cache CPU.
  4. Harga sangat mahal dan kapasitas kecil.

DRAM (Dynamic RAM)

  1. Lebih murah karena memakai struktur 1 transistor + 1 kapasitor.
  2. Perlu refresh berkala.
  3. Menjadi standar main memory komputer.

Pada masa ini mulai muncul memori seperti DRAM 16 Kb, 64 Kb, 256 Kb, hingga menuju 1 MB.

Generasi ROM & EEPROM (1980–1990-an)

Untuk penyimpanan permanen, digunakan ROM

Jenis jenis ROM

  1. PROM – hanya bisa diprogram sekali.
  2. EPROM – bisa dihapus dengan sinar UV.
  3. EEPROM – bisa dihapus dan ditulis ulang secara elektrik.

Generasi DDR RAM (1990-an – Sekarang)
