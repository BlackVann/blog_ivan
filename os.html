Sejarah Sistem Operasi

  • Sejarah Sistem Operasi DOS

    DOS adalah singkatan dari Disk Operating System, adalah salah satu sistem operasi yang pertama kali digunakan di komputer pribadi (PC). Pada masanya, DOS menjadi tulang punggung dari banyak komputer karena mampu mengelola file dan menjalankan berbagai program. DOS adalah sistem operasi berbasis teks, yang artinya kamu harus mengetikkan perintah di prompt untuk mengoperasikannya.

    DOS pertama kali diperkenalkan pada awal tahun 1980-an oleh IBM dengan nama PC-DOS. Namun, yang lebih dikenal oleh banyak orang adalah MS-DOS (Microsoft Disk Operating System) yang dikembangkan oleh Microsoft. MS DOS menjadi sangat populer karena digunakan di banyak PC di seluruh dunia.

    Seiring berjalannya waktu, MS-DOS terus mengalami perkembangan dengan berbagai versi yang menambahkan fitur dan kemampuan baru, seperti dukungan untuk hard disk, direktori, serta utilitas tambahan untuk pengelolaan sistem. Versi-versi awal hanya mampu menjalankan satu program pada satu waktu dan memiliki tampilan yang sepenuhnya berbasis teks tanpa antarmuka grafis.

    Namun, pada pertengahan tahun 1990-an, popularitas DOS mulai menurun setelah Microsoft merilis Windows 95, yang membawa antarmuka grafis (GUI) dan mampu menjalankan banyak program secara bersamaan (multitasking). Meskipun Windows 95 dan beberapa versi setelahnya masih menggunakan DOS sebagai dasar sistem booting, perannya semakin berkurang dari waktu ke waktu.

    Akhirnya, MS-DOS resmi dihentikan pada tahun 2000 dengan versi terakhirnya MS-DOS 8.0, yang disertakan dalam Windows ME (Millennium Edition). Sejak saat itu, Windows menjadi sistem operasi utama yang menggantikan DOS sepenuhnya, meskipun beberapa perintah DOS masih tetap dikenal dan digunakan melalui Command Prompt (CMD) di sistem Windows modern.

  • Sejarah Sistem Operasi Windows

    OS ini pertama kali diperkenalkan oleh Microsoft pada tahun 1985 dan sejak saat itu telah mengalami banyak perkembangan dan evolusi. Windows 1.0, versi pertama dari OS ini, dirilis pada November 1985. OS ini adalah sistem operasi grafis pertama yang dirancang untuk penggunaan komersial dan dapat digunakan pada PC yang menggunakan processor Intel 8086 atau 8088. Windows 1.0 memiliki tampilan antarmuka yang sederhana dan menawarkan dukungan untuk beberapa aplikasi, seperti Paintbrush, Write, dan Calculator. Setelah perilisan Windows 1.0, Microsoft terus melakukan pengembangan untuk meningkatkan kemampuan dan tampilan sistem operasinya.

    Pada Desember 1987, Microsoft merilis Windows 2.0, yang membawa peningkatan signifikan dalam hal tampilan dan fungsi. Versi ini sudah mendukung penggunaan tampilan jendela yang bisa ditumpuk (overlapping windows) dan penggunaan keyboard shortcut, serta mulai banyak digunakan bersama dengan aplikasi terkenal seperti Microsoft Word dan Excel.

    Kemudian pada Mei 1990, hadir Windows 3.0, yang menjadi terobosan besar bagi Microsoft. Sistem ini memiliki tampilan grafis yang jauh lebih menarik dan mendukung multitasking, sehingga banyak digunakan oleh pengguna rumahan maupun bisnis. Windows 3.1 yang dirilis pada 1992 memperbaiki stabilitas dan memperkenalkan TrueType Font, memungkinkan pengguna untuk menampilkan teks dengan kualitas lebih baik.

    Selanjutnya, pada 1995, Microsoft merilis Windows 95, yang menjadi tonggak penting dalam sejarah komputer pribadi. Windows 95 memperkenalkan menu Start, taskbar, dan konsep “plug and play”, yang membuat penggunaan komputer menjadi jauh lebih mudah. Windows 95 juga merupakan sistem pertama yang benar-benar mengintegrasikan MS-DOS dan antarmuka grafis Windows menjadi satu kesatuan.

    Setelah itu, muncul berbagai versi lanjutan seperti Windows 98 (1998), Windows ME (2000), Windows 2000 (2000), dan Windows XP (2001). Di antara semuanya, Windows XP menjadi salah satu versi paling populer karena tampilannya yang modern dan kestabilannya yang tinggi.

    Kemudian hadir Windows Vista (2007) dengan tampilan visual baru bernama Aero, diikuti oleh Windows 7 (2009) yang dianggap sebagai versi penyempurnaan dan sangat disukai banyak pengguna.

    Selanjutnya, Microsoft merilis Windows 8 (2012) yang memperkenalkan tampilan berbasis tile dan fokus pada layar sentuh, namun tidak terlalu diterima oleh banyak pengguna. Hal ini kemudian diperbaiki lewat Windows 10 (2015), yang menggabungkan keunggulan Windows 7 dan Windows 8 dengan tampilan yang lebih modern dan fitur keamanan yang lebih kuat.

    Versi terbaru, Windows 11, diluncurkan pada Oktober 2021, membawa tampilan antarmuka yang lebih halus, ikon tengah seperti macOS, dukungan aplikasi Android, serta peningkatan pada performa dan keamanan sistem.
